# 📜 Thai Constitution Post-Processor (Stage 2)

สคริปต์สำหรับประมวลผลข้อมูลรัฐธรรมนูญไทยจากรูปแบบ Raw JSON (ที่ได้จาก Pipeline การสกัดข้อความด้วย PDFPlumber/Markdown) ให้กลายเป็น **Structured JSON** และ **CSV** ที่มีโครงสร้างชัดเจน พร้อมสำหรับการนำไปวิเคราะห์ข้อมูล (Data Analysis) หรือทำ Visualization ต่อไป

## ✨ สิ่งที่ปรับปรุงจากเวอร์ชันเดิม (What's New)

1.  **Data Cleaning:**
    * ตัด Header/Footer ของราชกิจจานุเบกษา (เล่ม, หน้า, วันที่) ออก
    * แก้ปัญหาการแสดงผล "สระอำ" ผิดเพี้ยน (เช่น `จ านวน` → `จำนวน`)
    * แปลงตัวเลขไทยทั้งหมดให้เป็นเลขอารบิก (Normalize)
2.  **Structural Parsing:**
    * ยุบรวมข้อความจากทุกหน้า (Pages) ให้เป็นเนื้อหาเดียวกัน (Full Text)
    * แยก **คำปรารภ (Preamble)** ออกจากเนื้อหาหลัก
    * จัดโครงสร้างข้อมูลใหม่โดยแบ่งตาม **"หมวด" (Chapter)** และสกัดแยกแต่ละ **"มาตรา" (Section)** อย่างชัดเจน
3.  **Data Export:**
    * สร้างไฟล์ CSV รูปแบบ Flat Table สำหรับการวิเคราะห์รายมาตรา หรือเปรียบเทียบรัฐธรรมนูญหลายฉบับ (Cross-version analysis)

## 🏗️ โครงสร้างข้อมูลที่เปลี่ยนแปลง (JSON Structure)

### ❌ ของเดิม (Raw Data)
โครงสร้างเดิมจะเก็บข้อมูลแยกตาม **หน้า (Pages)** ซึ่งยากต่อการค้นหาหรืออ้างอิงเป็นรายมาตรา
```json
{
  "id": "const_2475",
  "year_th": 2475,
  "pages": [
    {
      "page_number": 1,
      "raw_markdown": "หน้า ๕๒๙\nราชกิจจานุเบกษา...\nหมวดที่ 1\nมาตรา 1..."
    }
  ]
}
```

### ✅ ของใหม่ (Structured Data)
ข้อมูลถูกจัดระเบียบใหม่ตาม **โครงสร้างทางกฎหมาย (หมวด และ มาตรา)** และเพิ่ม `summary` เพื่อสรุปสถิติ
```json
{
  "id": "const_2475",
  "year_th": 2475,
  "preamble": "พระบาทสมเด็จพระปรมินทรมหาประชาธิปก...",
  "summary": {
    "total_chapters": 7,
    "total_sections": 68
  },
  "chapters": [
    {
      "chapter_number": 1,
      "chapter_title": "บททั่วไป",
      "sections": [
        {
          "section_number": 1,
          "text": "อำนาจสูงสุดของประเทศนั้น เป็นของราษฎรทั้งหลาย"
        }
      ]
    }
  ]
}
```

## 📊 โครงสร้างไฟล์ CSV (Export Format)

การ Export เป็น CSV จะถูกแปลงให้อยู่ในรูปแบบ **Flat Table (1 แถว = 1 มาตรา)** เพื่อให้ง่ายต่อการนำไปใช้ใน Pandas, Excel หรือ BI Tools โดยมีคอลัมน์ดังนี้:

| คอลัมน์ | รายละเอียด | ตัวอย่าง |
| :--- | :--- | :--- |
| `constitution_id` | รหัสอ้างอิงรัฐธรรมนูญ | `const_2475` |
| `year_th` | ปี พ.ศ. | `2475` |
| `year_ce` | ปี ค.ศ. | `1932` |
| `name_short` | ชื่อเรียกสั้น | `Constitution 2475` |
| `era` / `regime_type` | ยุคสมัย / รูปแบบการปกครอง | *(ดึงจาก metadata เดิม)* |
| `chapter_number` | เลขหมวด (0 = บททั่วไป, -1 = บทเฉพาะกาล) | `1` |
| `chapter_title` | ชื่อหมวด | `บททั่วไป` |
| `section_number` | เลขมาตรา | `1` |
| `section_text` | เนื้อหาของมาตรานั้น ๆ | `อำนาจสูงสุดของประเทศนั้น...` |

*(ระบบจะสร้างไฟล์ `sections_XXXX.csv` สำหรับแต่ละฉบับ และไฟล์ `all_sections_combined.csv` ที่รวมทุกฉบับไว้ด้วยกัน)*

## 🚀 วิธีใช้งาน (Usage)

โครงสร้างโฟลเดอร์สำหรับรันผ่าน Jupyter Notebook (`constReStruc.ipynb`):
* นำไฟล์ Raw JSON ต้นฉบับไปวางไว้ในโฟลเดอร์: `data/`
* รันโค้ดใน Notebook ทั้งหมด
* ผลลัพธ์ (Structured JSON และ CSV) จะถูกบันทึกไว้ในโฟลเดอร์: `struc-data/`
